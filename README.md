## Getting Started

### 安装ansu_ui

#### Android

Change the minimum Android sdk version to 21 (or higher) in your `android/app/build.gradle` file.

```gradle
minSdkVersion 21
```

## 贡献

### 组件命名

* 文件命名规则 `as_***`,例如 `as_button`
* 类名命名规则 `AS***`,例如`ASButton`
